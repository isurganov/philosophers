<!-- GETTING STARTED -->
## Getting Started

First of all clone my repo with ```git clone```.

### Run it

Some steps how to run this minishell.
* run Makefile
  ```sh
  make 
  ```
* clean obj files
  ```sh
  make clean
  ```
* clean obj + binary
  ```sh
  make fclean
  ```
* fclean all
  ```sh
  make re
  ```
